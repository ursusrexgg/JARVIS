# Rules

Gebruik deze map om `AGENTS.md` kort te houden. `AGENTS.md` routeert; `Rules/` bevat de detailafspraken.

## Regelbestanden

- `communication-style.md`: communicatie- en antwoordstijl.
- `file-management.md`: hoe bestanden worden gemaakt, verplaatst en gearchiveerd.
- `privacy-and-publishing.md`: privacy, publicatie en externe deling.
