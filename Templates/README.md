# Templates

Deze map bevat rootbrede templates voor Jarvis.

Gebruik deze templates voor terugkerende outputvormen die niet specifiek bij een workflow horen.

## Templates

- `session-summary.md`: afsluiting van een werksessie.
- `onboarding-questions.md`: interviewvragen om de contextlaag te vullen.
- `project-readme.md`: basisformat voor nieuwe projectmappen.

## Werkafspraak Voor Codex

Gebruik templates alleen wanneer de taak daarom vraagt of wanneer een output anders telkens opnieuw moet worden bedacht.
