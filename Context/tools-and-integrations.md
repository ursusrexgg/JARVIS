# Tools And Integrations

## Doel

Dit bestand houdt bij welke tools, repositories en externe koppelingen relevant zijn voor Jarvis.

## Huidige Tools

### Codex

- Primaire assistantomgeving.
- Jarvis is ingericht voor gebruik met Codex en `AGENTS.md`.

### GitHub

- Rootrepo: `https://github.com/ursusrexgg/JARVIS`
- Bestaande onderdelen zijn gekoppeld als submodules.

### Obsidian

- Wordt gebruikt of voorbereid als dashboard- en kennisomgeving.
- Dashboard is gewenst, maar nog niet gebouwd.
- Heeft de voorkeur als mogelijke plek voor projectmanagement, mits projecten, deadlines en overzicht daar simpel genoeg in te richten zijn.

### Google Workspace

- Google Drive wordt gebruikt.
- Gmail wordt gebruikt.
- Google Calendar wordt gebruikt voor beschikbaarheid en agenda.

### Visual Studio Code / Codex

- Arthur werkt met VS Code en Codex.

### Canva

- Arthur is bekend met Canva.

### Voice Inc.

- Wordt gebruikt voor speech-to-text.

## Projectmanagement Wens

- Moet meerdere projecten kunnen beheren.
- Moet onderscheid kunnen maken tussen priveprojecten en gedeelde projecten.
- Sereen moet mogelijk deelbaar zijn met Joas.
- Moet deadlines en simpele opvolging ondersteunen.
- Moet dagelijkse uitvoering ondersteunen: wat is vandaag belangrijk en wat kan Arthur afvinken?
- Moet per project voortgang, deadlines, onderliggende taken en taakbeschrijvingen zichtbaar maken.
- Voorkeur: geen extra tool als Obsidian voldoende is.
- Alternatief: eenvoudige, liefst gratis projectmanagementtool.

Voeg alleen concrete toolafspraken toe wanneer Arthur bevestigt dat ze gebruikt gaan worden.
