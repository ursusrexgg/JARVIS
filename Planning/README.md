# Planning

Deze map bevat nog maar een simpele takenlijst.

## Startbestand

- `tasks.md`: de enige actieve checklist met openstaande taken.

## Werkafspraak Voor Codex

Gebruik `Planning/tasks.md` wanneer Arthur vraagt:

- wat er nog openstaat;
- om taken toe te voegen;
- om taken samen te voegen;
- om een openstaande taak af te vinken.

Gebruik Obsidian niet als volledig projectmanagementsysteem. Projectcontext staat in `PROJECTS.md` en de projectmappen; echte projectmanagementtaken worden later in een externe tool beheerd.
