# Archief

Deze map bevat materiaal dat niet meer actief is, maar wel bewaard moet blijven.

## Werkafspraak

Verwijder liever niets definitief wanneer het om Jarvis-structuur, projectcontext, workflows, besluiten of referentiemateriaal gaat.

Als iets niet meer actief is:

1. verplaats het naar een passende archieflocatie;
2. noteer waar het vandaan kwam als dat later nuttig is;
3. verwijder alleen definitief als Arthur dat expliciet vraagt.

## Mogelijke Submappen

- `Projecten/`
- `Workflows/`
- `Context/`
- `Referenties/`

Deze submappen kunnen later worden toegevoegd zodra er echt iets te archiveren is.
