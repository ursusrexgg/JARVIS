# Morning Review

## Wanneer Gebruiken

Gebruik deze skill wanneer Arthur vraagt om:

- dagplanning;
- ochtendreview;
- focus voor vandaag;
- prioriteiten bepalen;
- "wat moet ik nu doen?"

## Benodigde Context

Lees gericht:

1. `../Planning/tasks.md`
2. `../Context/current-priorities.md`
3. `../PROJECTS.md`

Lees alleen extra projectbestanden als de dagplanning om dat project draait.

## Stappen

1. Vat de actuele prioriteiten kort samen.
2. Benoem maximaal drie focusblokken voor vandaag.
3. Markeer openstaande taken die vandaag aandacht vragen.
4. Stel een realistische volgorde voor.
5. Vraag alleen om verduidelijking als planning onmogelijk is zonder extra informatie.

## Output

Geef een compacte dagplanning met:

- belangrijkste uitkomst;
- top 3 acties;
- optionele focusblokken;
- dingen die bewust niet vandaag hoeven.

## Stopregels

Plan geen externe agenda-afspraken zonder expliciete opdracht.

Gebruik geen Gmail, kalender of externe tools tenzij Arthur daar expliciet om vraagt.
