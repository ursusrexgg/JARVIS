# Skills

Deze map bevat lokale Jarvis-skills: compacte proceskaarten voor terugkerende executive-assistant taken.

Ze zijn bedoeld voor Codex-gebruik binnen deze workspace. Dit zijn geen externe plugins en geen systeem-skills.

## Beschikbare Skills

- `morning-review.md`: dagstart en focus bepalen.
- `project-pulse-check.md`: projecten kort nalopen op status, risico en volgende actie.
- `decision-capture.md`: structurele keuzes vastleggen.
- `research-brief.md`: researchvraag scherp maken en output structureren.
- `content-repurpose.md`: bestaande input omzetten naar contentvarianten.

## Format

Elke skill bevat:

- wanneer gebruiken;
- benodigde context;
- stappen;
- output;
- stopregels.

## Werkafspraak Voor Codex

Gebruik een skill alleen wanneer de taak daar duidelijk bij past of wanneer Arthur erom vraagt.

Lees niet automatisch alle skills. Kies de relevante skill via `ROUTING.md` of deze README.
