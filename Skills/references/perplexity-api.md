# Perplexity API

## Gebruik Binnen Jarvis

Gebruik Perplexity via API voor gericht actueel onderzoek, vooral productvergelijkingen, toolselectie en marktvragen.

## Belangrijke Punten

- API-key hoort lokaal in `PERPLEXITY_API_KEY`.
- Zet de API-key nooit in Git, Jarvis-bestanden of chatberichten.
- Sonar API is geschikt voor web-grounded antwoorden met bronnen.
- Perplexity ondersteunt een OpenAI-compatible Chat Completions formaat.
- Standaardmodel voor productonderzoek: `sonar-pro`.
- Voor zware research kan `sonar-deep-research` gebruikt worden, maar dit kan duurder/langzamer zijn.

## Bronnen

- Sonar quickstart: https://docs.perplexity.ai/docs/sonar/quickstart
- Search quickstart: https://docs.perplexity.ai/docs/search/quickstart
