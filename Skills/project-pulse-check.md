# Project Pulse Check

## Wanneer Gebruiken

Gebruik deze skill wanneer Arthur vraagt om:

- projectstatus;
- pulse check;
- voortgangsoverzicht;
- risico's;
- eerstvolgende acties per project.

## Benodigde Context

Lees gericht:

1. `../PROJECTS.md`
2. `../Projecten/README.md`
3. relevante project-README's
4. `../Context/current-priorities.md`
5. `../Planning/tasks.md`

## Stappen

1. Maak een lijst van relevante projecten.
2. Bepaal per project: status, laatste bekende context, risico/open vraag, volgende actie.
3. Scheid actieve projecten van wachtende projecten.
4. Benoem waar informatie ontbreekt.
5. Stel maximaal vijf concrete vervolgstappen voor.

## Output

Gebruik dit format:

```markdown
## Project Pulse

### Actief

### Wacht

### Risico's / Open Vragen

### Aanbevolen Volgende Acties
```

## Stopregels

Wijzig projectstatussen niet zonder duidelijke opdracht.

Verplaats projecten niet zonder aparte bevestiging.
