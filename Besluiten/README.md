# Besluiten

Deze map bevat structurele beslissingen over Jarvis, projecten, workflows en werkwijze.

Gebruik dit niet voor gewone taken. Gebruik het alleen wanneer een keuze later belangrijk kan zijn om terug te begrijpen.

## Startbestanden

- `decision-log.md`: chronologisch overzicht van belangrijke beslissingen.
- `decision-template.md`: format voor nieuwe beslissingen.

## Wanneer Een Besluit Vastleggen

Leg een besluit vast wanneer het gaat over:

- structuur van Jarvis;
- projectrichting;
- workflowregels;
- privacy of publicatie;
- tooling of GitHub;
- terugkerende werkwijze;
- keuzes die later discussie of verwarring kunnen voorkomen.

## Werkafspraak Voor Codex

Vraag eerst kort of een besluit in het log moet wanneer Arthur een structurele keuze maakt. Leg niets als definitief besluit vast zonder bevestiging of duidelijke opdracht.
