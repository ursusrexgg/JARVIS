# Notitie Inbox

Gebruik deze map voor snelle ideeën, ruwe gedachten en losse notities die nog geen vaste plek hebben.

## Bestanden

- `snelle-ideeen.md` — korte ideeën, ingevingen, zinnen, hooks, onderwerpen.
- `losse-notities.md` — langere ruwe notities die nog niet gestructureerd zijn.

## Verwerken

Verwerk periodiek naar:

- `../schrijven/` voor schrijfoutput;
- `../meetings/` voor gespreksnotities;
- `../reflecties/` voor persoonlijke reflectie;
- `../uitgewerkt/` voor aangescherpte notities;
- `../../Kennis/` wanneer het blijvende kennis wordt.
