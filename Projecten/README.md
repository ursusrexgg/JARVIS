# Projecten

Deze map bevat concrete initiatieven waar Arthur aan werkt.

Gebruik `PROJECTS.md` in de root als lichte statuskaart. Gebruik de mappen hieronder voor projectcontext, documenten en projectspecifieke output.

## Actieve Projectmappen

- `Sereen/`: product/project met compliance-documentatie.
- `Personal Brand Arthur/`: project in opbouw voor Arthur's eigen personal brand.
- `Jarvis Dashboard/`: project in opbouw voor de visuele daily-driver app van Jarvis.

## Werkafspraak

Een projectmap krijgt minimaal:

- een `README.md`;
- doel en status;
- belangrijkste documenten;
- eerstvolgende acties;
- stopregels als er gevoelige inhoud of externe publicatie bij komt kijken.
