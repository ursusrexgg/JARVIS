# Handoffs

Deze map bevat korte afsluitnotities per werksessie.

Gebruik de nieuwste handoff aan het begin van een volgend gesprek om snel te zien:

- wat er is gedaan;
- welke besluiten zijn genomen;
- wat openstaat;
- wat de meest logische volgende stap is.

## Werkafspraak

- Maak een nieuwe handoff wanneer een sessie inhoudelijk wordt afgerond.
- Houd het kort en praktisch.
- Zet alleen blijvende context in andere contextbestanden wanneer Arthur dat expliciet wil.
- Gebruik handoffs als overdracht, niet als vervanging van `Besluiten/decision-log.md`, `PROJECTS.md` of projectdocumentatie.
